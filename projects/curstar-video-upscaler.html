🎯 Problem

Low-resolution video footage — whether from older cameras, compressed streams, or surveillance systems — loses critical visual detail. Traditional upscaling methods like bicubic and Lanczos interpolation just blur pixels larger without reconstructing real detail.

I wanted to test whether diffusion-based super-resolution could reconstruct meaningful visual detail at the frame level, producing output that looks genuinely sharper rather than simply smoothed.

🔧 Approach

Implemented the STAR (Spatial-Temporal Augmented Resolution) diffusion model pipeline in Python. The system processes video frame-by-frame, applying a pretrained diffusion model to upscale each frame while maintaining temporal consistency.

Optimized for GPU-accelerated batch processing on an RTX 4090. The pipeline handles video decoding, frame extraction, model inference, and video re-encoding in a single automated workflow.

🧠 Key Decisions

Chose the STAR diffusion architecture over GAN-based upscalers (Real-ESRGAN) because diffusion models produce fewer artifacts on diverse content types. GANs can hallucinate textures that look plausible but aren't actually present in the source material — diffusion models tend to be more conservative and faithful.

Processed frames individually rather than using video-native models to maintain flexibility across different input formats and resolutions. This trades some temporal smoothness for broader compatibility and simpler pipeline architecture.

Core pipeline logic — frame extraction, diffusion inference, and video reassembly.

upscale.pyPython
def upscale_video(input_path, scale=4):
    frames = extract_frames(input_path)
    model = STARModel.from_pretrained("star-sr-x4")
    model.to("cuda").eval()

    upscaled = []
    for frame in tqdm(frames, desc="Upscaling"):
        tensor = to_tensor(frame).unsqueeze(0).cuda()
        with torch.no_grad():
            sr = model(tensor)
        upscaled.append(to_image(sr))

# Re-encode upscaled frames to output video
writer = cv2.VideoWriter(
    output_path, cv2.VideoWriter_fourcc(*"mp4v"),
    fps, (w * scale, h * scale)

for frame in upscaled:
    writer.write(cv2.cvtColor(frame, cv2.COLOR_RGB2BGR))
