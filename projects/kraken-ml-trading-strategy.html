🎯 Problem

Cryptocurrency markets generate massive amounts of OHLCV data but most retail trading strategies are based on lagging indicators or gut feel. I wanted to build a systematic, ML-driven approach that could identify short-term momentum signals from raw market data — tested on real Kraken exchange data.

The goal wasn't to build a get-rich-quick bot. It was to prove that a rigorous, end-to-end ML pipeline — from data ingestion through feature engineering, model training, and cloud deployment — could be built by one person using open-source tools, producing measurably better-than-random directional predictions.

🔧 Approach

Built a feature engineering pipeline that transforms raw candlestick data into 20+ technical features (RSI, MACD, Bollinger Band width, volume momentum, etc.). Trained gradient boosted classifiers using scikit-learn with walk-forward validation to avoid lookahead bias.

Deployed the model on Google Cloud Functions to generate signals on a schedule via the Kraken REST API. The pipeline runs serverless — no VM to maintain, automatic scaling, and cost-efficient execution for periodic inference.

🧠 Key Decisions

Chose scikit-learn over deep learning because tabular features + gradient boosting consistently outperform neural nets on structured financial data at this scale. The research literature backs this up — XGBoost/LightGBM dominate Kaggle tabular competitions for a reason.

Used walk-forward validation instead of random splits to respect the time-series nature of the data. Random cross-validation on time-series data creates lookahead bias and produces artificially inflated metrics that collapse in live deployment.

Deployed on GCP Cloud Functions for serverless execution rather than maintaining a VM. This keeps costs near-zero during development while providing a production-ready deployment path.

def engineer_features(df):
    # Technical indicators from raw OHLCV
    df['rsi'] = ta.rsi(df['close'], length=14)
    df['macd'] = ta.macd(df['close'])['MACD_12_26_9']
    df['bb_width'] = ta.bbands(df['close'])['BBB_5_2.0']
    df['vol_sma'] = df['volume'].rolling(20).mean()
    return df.dropna()
